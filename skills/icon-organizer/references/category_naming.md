# Category Naming Convention

This document describes the standard naming conventions for cloud product icon category folders.

## Naming Pattern

Categories should follow this naming pattern:
```
{NN} {Category Name}
```

Where:
- `{NN}` is a two-digit number with leading zero (01, 02, ..., 17)
- `{Category Name}` is the category name in Title Case

## Best Practices

1. Always use numeric prefixes to ensure consistent ordering
2. Use Title Case for English category names
3. Keep category names concise but descriptive
4. Group related products together in the same category
5. Refer to the official cloud vendor's product categorization when in doubt
